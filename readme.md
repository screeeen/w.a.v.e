Wave is a visual software that reacts to music.
**controls table**
|key |function | |
|----------------|-------------------------------|-----------------------------|
|Z|`show commands overlay`  
|1 |`changes 1/3 shapes`|
|s |`size`|
|d |`randomize`|
|f |`speed and frequency`|
|g |`change direction on rotation`|
|r |`rotation mode`|

include minim to run
CMD+SHIFT+B to launch from vscode
